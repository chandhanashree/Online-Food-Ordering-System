# Online Food Ordering System

## Overview
An online food ordering system developed using PHP and MySQL that allows customers to browse menus, place orders, track deliveries, and download invoices.

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- JavaScript
- AJAX
- jQuery

## Features

### Customer Module
- Registration and Login
- Browse Food Menu
- Add to Cart
- Place Orders
- Track Orders
- Download Invoice

### Admin Module
- Dashboard
- Manage Food Categories
- Manage Food Items
- Manage Users
- Manage Orders
- Sales Reports

## Database
- MySQL
- Database file: fosdb.sql
